# 知识导图

## 目标：

一个更好的学习平台， 对思维导图进行提升。

加入进度控制（知识点掌握进度）、笔记、答疑、练习等功能， 让思维导图互联网化！

## 后台：

基于 openshift(免费) + python + flask.

数据库定为 PostgreSQL 9.2 + redis

## 前端

1. 开源的JS思维导图项目
2. tryregex
3. elevator saga
4. d3.js 做的思维导图
5. markdown-it 进行markdown解析

一个全面、多方位的学习平台
